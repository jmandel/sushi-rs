{% raw %} US Core Implementation Guide

The US Core Implementation Guide defines the minimum set of FHIR RESTful interactions for each of the US Core Profiles to access patient data. The Data Access Framework and the Argonaut DSTU2 guides initially formed the basis for US Core. The Profiles are updated annually to support the latest requirements from The Office of the National Coordinator for Health Information Technology (ONC) U.S. Core Data for Interoperability (USCDI). For a summary of significant changes and a detailed list of the changes between versions, see the Change Notes at https://hl7.org/fhir/us/core/changes.html
