Title:Common Period-Tracking Symptoms
Status:Draft as of 2026-07-03 02:41:36-0500
Definition:

A small, non-normative starter set of period-tracking symptoms that applications commonly record, offered so implementers can bootstrap a consistent symptom vocabulary. It includes SNOMED CT concepts where the fit is exact enough. It is NOT a closed or required binding: a fact's symptom value MAY use another SNOMED CT concept when the meaning is exact.
