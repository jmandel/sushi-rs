rust_sushi + snapshot_gen, in the browser

WASM P0 spike — compiles the cycle IG (FSH → FHIR) and generates validation-grade snapshots entirely client-side via WASI, no server compute.

Click “Run” to compile the cycle IG and generate snapshots in this tab.
Both binaries run against an in-memory virtual filesystem (cycle sources +
a minimal FHIR package cache subset). Outputs are hashed for the native
byte-match gate — see the downloadable wasm-hashes.json.
